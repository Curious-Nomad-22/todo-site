# Todo Website (Full Stack)

This is a full-stack Todo web application with user authentication.

Users can:
- Register with a username and password
- Log in securely
- Create, view, and manage their personal todo list
- Return later and see their saved todos

---

## Tech Stack

### Backend
- FastAPI
- SQLModel / SQLAlchemy
- PostgreSQL
- JWT Authentication

### Frontend
- HTML
- CSS
- JavaScript (Vanilla)

### Deployment
- Backend: Render (Web Service)
- Frontend: Render (Static Site)
- Database: Render PostgreSQL
